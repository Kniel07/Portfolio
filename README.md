# Kirk Ranches — Portfolio

Static single-page portfolio site. Plain HTML/CSS/JS, no build step, deployed via GitHub Pages.

## Structure
- `index.html` — page structure and content
- `css/styles.css` — design system (colors, type, layout)
- `js/script.js` — boot sequence, scroll-spy nav, animated metrics, scroll reveal
- `assets/Kirk_Ranches_Resume.pdf` — downloadable résumé

## Local preview
Just open `index.html` in a browser, or serve the folder with any static file server.
